# 📝 Registration Form

A simple and responsive **Registration Form** built using **HTML, CSS, Bootstrap 5, and JavaScript**. This project validates user input for Name, Phone Number, Email, and Password using JavaScript and Regular Expressions (Regex).

## 📋 Validation Rules

### 👤 Name

- Required
- Only lowercase letters are allowed

Example:

```
rifat ✅
Rifat ❌
rifat123 ❌
```

---

### 📱 Phone Number

- Required
- Must contain exactly **11 digits**

Example:

```
01712345678 ✅
0171234567 ❌
017123456789 ❌
```

---

### 📧 Email

- Required
- Must be a valid email address

Example:

```
rifat@gmail.com ✅
user123@yahoo.com ✅
rifatgmail.com ❌
```

---

### 🔒 Password

- Required
- Minimum 8 characters
- Must contain:
  - One uppercase letter
  - One lowercase letter
  - One number
  - One special character

Example:

```
Rifat@123 ✅
rifat123 ❌
Rifat123 ❌
```
